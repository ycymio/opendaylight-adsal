Notifications Summary

Name Description
node-connector-removed A notification sent by someone who realized there was a node-connector was removed, but did not modify the data tree. Describes that a node-connector has been removed but is for whatever reason is not modifying the data tree. Listeners interested update changes should register a data change listener for notifications on removals.
